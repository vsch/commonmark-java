## GfmIssues

Converts GitHub `#issue` syntax to GitHubIssue nodes that renders as a link to the issue

Only digits allowed

```````````````````````````````` example GfmIssues: 1
abc #123abc xyz
.
<p>abc #123abc xyz</p>
.
Document[0, 16]
  Paragraph[0, 16]
    Text[0, 15] chars:[0, 15, "abc # … c xyz"]
````````````````````````````````


```````````````````````````````` example GfmIssues: 2
abc #123-4 xyz
.
<p>abc #123-4 xyz</p>
.
Document[0, 15]
  Paragraph[0, 15]
    Text[0, 14] chars:[0, 14, "abc # … 4 xyz"]
````````````````````````````````
