## Text

Converts HTML to markdown

```````````````````````````````` example Text: 1
Expected rendered HTML

.
<p>Expected rendered HTML</p>
````````````````````````````````


```````````````````````````````` example Text: 2
Expected rendered HTML  
Another line

.
<p>Expected rendered HTML<br>Another line</p>
````````````````````````````````


```````````````````````````````` example Text: 3
Expected rendered HTML  

Another line

.
<p>Expected rendered HTML<br><br>Another line</p>
````````````````````````````````
